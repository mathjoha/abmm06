Project Overview

This is a Quarto website for ABMM06, a Swedish university course on quantitative methods for ALM (Archive, Library, Museum) master’s students. The course covers surveys, digital methods, and statistical concepts.

Build Commands

# Preview site with live reload
quarto preview

# Build the site
quarto render

# Render a specific file
quarto render slides/kvantitativa-metoder.qmd

Architecture

  • Quarto website project configured in _quarto.yml
  • Output directory: _site/ (generated, gitignored)
  • Language: Swedish (lang: sv)

Content Structure

  • slides/ - RevealJS presentations (.qmd files with reveal format)
  • lessons/ - Detailed lesson content
  • dashboards/ - Interactive tools using Observable JS (OJS)
  • index.qmd - Course homepage
  • about.qmd - Course information

Interactive Components

Dashboards use embedded Observable JavaScript ({ojs} code blocks) for interactive visualizations: - Probability simulations - Statistical calculators - Data type explorers

Content Guidelines

  • All content is in Swedish
  • Use Swedish/English terminology table format for key concepts (see index.qmd)
  • Slides use RevealJS format via Quarto

# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a Quarto website for ABMM06, a Swedish university course on quantitative methods for ALM (Archive, Library, Museum) master's students. The course covers surveys, digital methods, and statistical concepts.

## Build Commands

```bash
# Preview site with live reload
quarto preview

# Build the site
quarto render

# Render a specific file
quarto render slides/kvantitativa-metoder.qmd
```

## Architecture

- **Quarto website project** configured in `_quarto.yml`
- **Output directory**: `_site/` (generated, gitignored)
- **Language**: Swedish (lang: sv)

### Content Structure

- `slides/` - RevealJS presentations (`.qmd` files with reveal format)
- `lessons/` - Detailed lesson content
- `dashboards/` - Interactive tools using Observable JS (OJS)
- `index.qmd` - Course homepage
- `about.qmd` - Course information

### Interactive Components

Dashboards use embedded Observable JavaScript (`{ojs}` code blocks) for interactive visualizations:
- Probability simulations
- Statistical calculators
- Data type explorers

## Content Guidelines

- All content is in Swedish
- Use Swedish/English terminology table format for key concepts (see index.qmd)
- Slides use RevealJS format via Quarto
